# 📧 TELC A2·B1 – Schreiben Teil 31
## E-Mail schreiben: Format + 10 Übungen

---

## ⭐ WICHTIG: Das müssen Sie wissen

| Punkt | Info |
|-------|------|
| ⏱️ Zeit | 30 Minuten |
| 📝 Wörter | ca. 100–130 Wörter |
| 📌 Aufgabe | Eine E-Mail schreiben und alle Fragen beantworten |
| ✅ Punkte bekommen | Alle Inhaltspunkte + gute Grammatik (B1) |

> **Regel Nr. 1:** Es ist wichtiger, **alle Punkte zu beantworten**, als perfekte Grammatik zu schreiben!

---

## 📋 DAS RICHTIGE FORMAT (immer gleich!)

```
Von:    [Ihr Name]@email.de
An:     [Name der anderen Person]@email.de
Betreff: AW: [Thema der E-Mail]

[ANREDE]
Liebe/r [Vorname],          ← für Freunde / informell (du)
Sehr geehrte/r Frau/Herr [Nachname],  ← für formell (Sie)

[EINLEITUNG – 1 Satz]
Vielen Dank für deine/Ihre E-Mail.
Ich freue mich sehr, dass du/Sie mir geschrieben hast/haben.

[HAUPTTEIL – alle Fragen beantworten!]
Punkt 1: ... (weil / dass / wenn)
Punkt 2: ... (Außerdem / Ich würde gerne...)
Punkt 3: ... (damit / obwohl / weil)
Punkt 4: ... (Ich freue mich auf...)

[SCHLUSS – 1 Satz]
Ich freue mich auf deine/Ihre Antwort.
Bis bald! / Viele Grüße!

[GRUß]
Liebe Grüße,         ← informell
Mit freundlichen Grüßen,  ← formell

[IHR NAME]
```

---

## 🔑 B1-GRAMMATIK: Diese 6 Wörter geben Extra-Punkte!

| Wort | Bedeutung | Beispiel |
|------|-----------|---------|
| **weil** | because | *Ich komme gerne, **weil** es interessant ist.* |
| **dass** | that | *Ich freue mich, **dass** du kommst.* |
| **wenn** | when / if | ***Wenn** das passt, bin ich dabei.* |
| **damit** | so that | *Ich schreibe früh, **damit** du planen kannst.* |
| **obwohl** | although | *Ich komme, **obwohl** ich wenig Zeit habe.* |
| **würde** | would | *Ich **würde** gerne helfen.* |

> ⚠️ **Achtung Wortstellung:** Nach diesen Wörtern kommt das Verb **am Ende!**
> ✅ *Ich komme, weil es interessant **ist**.*
> ❌ *Ich komme, weil es **ist** interessant.*

---

## 📚 NÜTZLICHE SÄTZE (auswendig lernen!)

### Einleitung (Anfang)
- *Vielen Dank für deine/Ihre E-Mail.*
- *Ich freue mich sehr, dass du/Sie mir geschrieben hast/haben.*
- *Es ist schön, von dir/Ihnen zu hören.*

### Zustimmung / Teilnahme
- *Ich komme sehr gerne.*
- *Ich nehme gerne teil.*
- *Ich würde mich sehr freuen.*

### Etwas mitbringen / helfen
- *Ich bringe gerne ... mit.*
- *Ich würde gerne ... helfen.*
- *Ich kann ... beisteuern.*

### Wie viele Personen
- *Ich komme alleine.*
- *Wir kommen zu zweit – mein/e ... und ich.*
- *Wir werden ... Personen sein.*

### Schluss (Ende)
- *Ich freue mich auf das Treffen / die Veranstaltung.*
- *Bis bald!*
- *Ich freue mich auf deine/Ihre Antwort.*

# 📝 10 ÜBUNGEN

> **Aufgabe immer gleich:**
> Lesen Sie die E-Mail. Schreiben Sie eine Antwort.

## ✅ CHECKLISTE (vor jeder Übung nutzen!)

Bevor Sie schreiben – **3 Minuten planen:**

- [ ] **Lesen Sie die E-Mail zweimal**
- [ ] **Markieren Sie alle Fragen/Punkte** (wie viele? 3 oder 4?)
- [ ] **Schreiben Sie auf**: *du* oder *Sie*?
- [ ] **Notieren Sie 2–3 Wörter** für jeden Punkt

Nach dem Schreiben – **prüfen:**

- [ ] Betreff geschrieben? (*AW: ...*)
- [ ] Richtige Anrede? (*Liebe/r* oder *Sehr geehrte/r*)
- [ ] **Alle Punkte** beantwortet?
- [ ] Mindestens **2x „weil", „dass" oder „wenn"** benutzt?
- [ ] Immer *du* ODER immer *Sie* – nie gemischt?
- [ ] Richtiger Gruß am Ende?
- [ ] Ca. **100–130 Wörter**?

## 🎯 GRAMMATIK-FEHLER VERMEIDEN

### ❌ A2 (zu einfach – gibt weniger Punkte)
> *Ich komme. Das ist gut. Ich bringe Salat mit. Das ist lecker.*

### ✅ B1 (besser – gibt mehr Punkte!)
> *Ich komme sehr gerne, **weil** ich solche Feste sehr mag. Ich bringe einen Salat mit, **der** immer gut ankommt.*
