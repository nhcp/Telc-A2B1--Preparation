## 3. Lesen — Reading

**Time:** 45 minutes | **Tasks:** 18 total (5 + 4 + 4 + 5)

### Teil 1 — Ads and notices (A2)
- Read **8 short ads** (a–h)
- Match **5 situations** to the correct ad
- **One situation has no match → write x**
- Goal: understand ads and match to situations (Kleinanzeigen verstehen)

**Strategy:**
- Read the situation carefully first — what exactly does the person need?
- Scan ads for key details: price, age limits, dates, location, what is offered
- Two ads will look similar — one tiny detail (age, day, purpose) separates them
- Eliminate clearly wrong options first

### Teil 2 — Short text, mixed tasks (A2/B1)
- Read one text (~150–200 words): email, announcement, or notice
- Answer **4 questions**: 1 true/false + 3 multiple choice
- Goal: understand detail information in a short text

**Strategy:**
- The wrong multiple choice options are often *true* about the text but don't answer the specific question
- Read the question first, then find the relevant part of the text
- For true/false: the whole statement must be correct — one wrong detail = falsch

### Teil 3 — Email matching (B1)
- Read **4 short emails**
- Match each to **1 of 8 subject lines** (Betreffzeilen)
- Goal: understand the global content/purpose of emails

**Strategy:**
- Ask: what is the *main purpose* of this email? (complaint, request, invitation, information, etc.)
- Do not get distracted by the topic — focus on what the sender wants to do

### Teil 4 — Longer text (B1)
- Read a longer article or text (~250–300 words)
- Answer **5 multiple choice questions**
- Goal: understand detail information in a longer text

**Strategy:**
- The text may include vocabulary above B1 — this is intentional; you don't need every word
- Each question relates to a specific part of the text — find that part, read it carefully
- 30 seconds per question maximum; if unsure, mark best guess and move on

### Recurring text topics
Work announcements, travel and holidays, housing, health, shopping, food, transport, education, community events, environment

## 4. Sprachbausteine + Lesen und Schreiben

**Time:** 35 minutes | **Tasks:** 10 + 2 + email writing

### Teil 1 — Grammar gap-fill (Sprachbausteine)
- One text (~150 words) with **10 gaps**
- Each gap has **3 options (a/b/c)** → choose the correct one
- Goal: understand text structures (grammar and connectors)

**Most tested grammar in Teil 1 (in order of frequency):**
1. Prepositions + correct case: *mit dem / auf den / seit einer / für einen*
2. Article endings (Nom/Akk/Dat): *der / die / das / den / dem / einem*
3. Verb forms: Perfekt, Präteritum, Konjunktiv II, modal verbs
4. Conjunctions: *weil, dass, wenn, als, ob, obwohl, bevor, nachdem*
5. Separable verb forms: *aufgemacht, angerufen, eingekauft*
6. Pronouns: personal, reflexive, relative

**Strategy:**
- Identify what the gap is testing before looking at the options
- If it's an article gap: check the gender of the noun + which case is required
- If it's a conjunction gap: check if the verb goes to the end in the following clause
- All 3 options may look grammatically possible — context decides

### Teil 2 — Read an email (Lesen und Schreiben)
- Read an incoming email (~80–120 words)
- Answer **2 questions**: 1 true/false + 1 multiple choice
- Goal: understand what the email says before replying

### Teil 3 — Write a reply email (Lesen und Schreiben)
- Write a reply to the email from Teil 2
- Address the specific content points given in the task
- Can be informal (friend/colleague) or semi-formal (organisation)
- Aim: ~80–100 words
- Goal: reply to a mail appropriately

**All required content points must be addressed — missing one loses marks.**

## 5. Hören + Hören und Schreiben

**Time:** 35 minutes (audio controls the pace — you cannot pause it)

> All pauses are built into the recording. If you miss something, don't panic — keep going.

### Teil 1 — Short everyday conversation, matching (A2)
- Hear a short conversation (~2–3 min)
- **5 matching tasks**: match information to options
- Heard **twice**
- Goal: understand information in a short everyday conversation

### Teil 2 — Professional conversation, true/false (A2/B1)
- Hear a conversation in a professional/work context
- **5 true/false tasks**
- Heard **twice**
- Goal: understand information in a professional conversation

### Teil 3 — Announcements, true/false (B1)
- Hear 5 short radio or public announcements
- **5 true/false tasks** — one per announcement
- **Heard once only ⚠️**
- Goal: understand information in announcements

### Teil 4 — Opinions on a topic, matching (B1)
- Hear people expressing opinions on a topic (e.g. 4–5 short clips)
- **5 matching tasks**: match opinion/statement to speaker
- Heard **twice**
- Goal: understand opinions and positions

### Teil 5 — Voicemail messages + write a note (B1)
- Hear voicemail messages
- **Write a note** with the key information
- Goal: understand answering machine messages and record information

### Schreiben part (inside Hören block)
- After Teil 5, write a short reply to a text/SMS message
- Answer the exact question asked — write more than one word
- Possible question words: *wann, was, warum nicht, welches, wie lange, wo, wohin*

### Listening strategies
- **Read all questions before the audio starts** — use every second of preparation time
- Write during the first listen, check during the second (where available)
- Use shorthand: *Mo 17:30 Arzt* not full sentences
- **Speakers often self-correct numbers** — always take the second/corrected value: *"um 18 Uhr — nein, um 18:30"*
- Teil 3 is heard **once only** — focus, do not overthink, go with first impression
- For true/false: the full statement must be correct — one wrong detail = falsch
- Focus on: numbers, times, names, places, reasons, days

## 6. Schreiben — Writing

**Time:** 10 minutes | **Task:** Write one short message

### The task
- You receive a short question or message (SMS, chat, note)
- Write a short reply — answer the question asked
- Write **more than one word** — use a full sentence to show your level
- Goal: respond appropriately to a question (Auf eine Frage angemessen reagieren)

> This is separate from the email writing in the Sprachbausteine block (Teil 3). The email writing happens earlier in the 35-minute block.

### Email writing (Teil 3 of Sprachbausteine block — done during the 35-min section)

**Structure to memorise:**

```
[Greeting]
Liebe/r [Name],                          ← informal
Sehr geehrte/r Frau/Herr [Name],         ← formal

[Acknowledge / thank]
Vielen Dank für deine/Ihre Nachricht.
Es freut mich, von dir/Ihnen zu hören.

[Address each content point — one sentence per point]
...

[Closing sentence]
Ich freue mich auf deine/Ihre Antwort.
Ich hoffe, bald von dir zu hören.

[Sign-off]
Viele Grüße / Liebe Grüße              ← informal
Mit freundlichen Grüßen                 ← formal
[Your name]
```

### Writing assessment criteria (Bewertungskriterien Schreiben)

| Criterion | What examiners check |
|-----------|---------------------|
| **Aufgabenbewältigung** | Did you address ALL required content points? |
| **Kommunikative Gestaltung** | Is the text coherent? Does it flow logically? Are connectors used? |
| **Korrektheit** | Grammar accuracy (word order, tenses, articles) |
| **Wortschatz** | Range and accuracy of vocabulary |

### Useful phrases

**Making requests (Konjunktiv II — essential for writing):**
- Könntest du / Könnten Sie mir bitte helfen?
- Ich würde mich freuen, wenn ...
- Wäre es möglich, dass ...?
- Es wäre schön, wenn wir uns treffen könnten.

**Giving reasons:**
- Ich kann leider nicht kommen, weil ich krank bin.
- Da ich arbeiten muss, habe ich keine Zeit.
- Ich habe leider keine Zeit, deshalb schreibe ich kurz.

**Agreeing / confirming:**
- Das klingt gut! Ich komme gerne.
- Ja, das passt mir sehr gut.
- Ich bin einverstanden.

**Declining / apologising:**
- Leider kann ich nicht kommen, weil ...
- Es tut mir leid, aber ich habe schon einen Termin.

**Asking for information:**
- Könntest du mir sagen, wann / wo / wie ...?
- Ich würde gerne wissen, ob ...

### Common email topics in the exam
- Accept or decline an invitation
- Respond to a complaint or problem
- Ask for information about a course, event, flat, job
- Write about a plan or appointment change
- Make a recommendation or give an opinion
- Thank someone and respond to their question

### Writing strategy (step by step)
1. Read the incoming email — **underline every content point** you must address
2. Write one sentence per content point — do not skip any
3. Check word order: verb in position 2 in main clause, verb at end after *weil/dass/wenn*
4. Open and close correctly — easy marks
5. Aim for 80–100 words — not 40, not 200

## 7. Sprechen — Speaking

**Time:** ~15 minutes | **Format:** Pair exam | **No preparation time**

Two examiners present: one conducts (Interlocutor), one assesses (Assessor).
If you arrive alone, one examiner takes the role of your partner.

### Teil 1A — Sich vorstellen / Introduce yourself (~2 min per person)

Prompt card on the table with key words — use them as a guide, not a script.

**Typical prompt words:** Name? | Alter? | Herkunft? | Wohnort? | Beruf/Studium? | Sprachen? | Hobbys? | Familie?

**Prepare this by heart — it is the only part you can fully memorise in advance:**
> Ich heiße [Name] und komme aus [Land]. Ich bin [Alter] Jahre alt und wohne seit [Zeit] in [Stadt/Ort]. Ich studiere [Fach] an der [Universität] / Ich arbeite als [Beruf]. In meiner Freizeit [Hobby]. Ich spreche [Sprachen] und lerne gerade Deutsch.

### Teil 1B — Anschlussfragen / Follow-up questions (~1 min per person)

Examiner asks deeper questions based on what you said.

**A2-level questions use:** Perfekt, wann/seit wann, warum, wo
**B1-level questions use:** Präteritum, Konjunktiv II, Passiv, modal verbs present + past

**Example B1 questions:**
- Seit wann lernen Sie Deutsch? Was hat sich seitdem verändert?
- Was haben Sie gestern gemacht?
- Was würden Sie gerne in Deutschland machen?
- Warum sind Sie nach Deutschland gekommen?
- Was müssen Sie nächste Woche tun?

**If you don't understand a question:**
> Könnten Sie die Frage bitte wiederholen? / Entschuldigung, ich habe das nicht ganz verstanden.

### Teil 2 — Gemeinsam etwas planen / Plan something together (~5 min)

You and your partner plan something together. Prompt cards with bullet points are provided.

- Make suggestions, react to partner's ideas, agree or politely disagree
- **Reaching final agreement is NOT required** — the conversation is what matters
- The examiners do NOT participate

**Typical scenarios:**
- Plan a leaving party for a colleague
- Choose furniture or equipment for a shared space
- Plan a team outing or office event
- Organise a birthday surprise
- Plan a community or club event

**Useful planning phrases:**

| Function | German phrases |
|----------|---------------|
| Suggest | Wir könnten ... / Wie wäre es mit ...? / Was hältst du von ...? / Ich schlage vor, dass wir ... |
| Agree | Das ist eine gute Idee. / Ja, das finde ich gut. / Einverstanden. / Das stimmt. |
| Disagree politely | Das finde ich nicht so gut, weil ... / Ich würde lieber ... / Ich bin nicht sicher, ob ... |
| Ask for opinion | Was meinst du dazu? / Wie siehst du das? / Hast du noch eine Idee? |
| Compromise | Vielleicht wäre es besser, wenn ... / Wir könnten auch ... / Was denkst du über ...? |
| Confirm | Also, wir machen ... / Dann sind wir uns einig. / Das klingt gut. |

### Teil 3 — Meinung begründen / Give and justify an opinion (~5 min)

You receive a question with two sides of a topic. ~1 minute to think and make notes.
Present your opinion with 2–3 reasons and examples. React to your partner's view.

**Typical opinion topics (from real exams and prep teachers):**
- Public transport vs. own car
- Working from home vs. in the office
- City life vs. countryside
- Fast food vs. cooking at home
- Social media: advantages vs. disadvantages
- Online learning vs. classroom learning
- Environment and recycling
- Part-time vs. full-time work
- Travelling alone vs. with others
- Smartphones: useful tool or problem?

**Useful opinion phrases:**

| Function | German phrases |
|----------|---------------|
| State opinion | Meiner Meinung nach ... / Ich finde, dass ... / Ich bin der Meinung, dass ... / Ich denke, dass ... |
| Give reason | ... weil / denn / da ... |
| Give example | Zum Beispiel ... / Ein gutes Beispiel dafür ist ... / Das zeigt, dass ... |
| Contrast | Auf der anderen Seite ... / Allerdings ... / Aber man muss auch sagen, dass ... |
| Agree with partner | Ich stimme dir zu, dass ... / Das sehe ich genauso. / Da hast du recht. |
| Disagree with partner | Das sehe ich etwas anders, weil ... / Ich bin nicht ganz einverstanden, denn ... |
| Conclude | Zusammenfassend kann ich sagen, dass ... / Deshalb denke ich, dass ... / Insgesamt finde ich ... |

### Speaking assessment criteria (Bewertungskriterien Sprechen)

| Criterion | What examiners look for |
|-----------|------------------------|
| **Aufgabenbewältigung** | Did you complete the task? Did you interact with your partner? |
| **Aussprache / Intonation** | Is your pronunciation understandable? Natural rhythm? |
| **Flüssigkeit** | Do you speak smoothly without long pauses? |
| **Wortschatz** | Do you use varied vocabulary, not just basic words? |
| **Satzstrukturen** | Do you use full sentences, conjunctions, subordinate clauses? |

### Real tips from people who passed

- **Interact — don't monologue.** Ask your partner questions, react: *"Ach wirklich? Das ist interessant. Was denkst du?"*
- **Length and interaction score higher than perfection.** A longer natural answer with small mistakes beats a short correct sentence.
- **Use connectors constantly:** *weil, deshalb, außerdem, trotzdem, obwohl* — they show B1 level
- **Self-correct out loud** if you notice a mistake — examiners value self-awareness
- **Vary your vocabulary:** instead of *gut* use *toll, prima, sinnvoll, praktisch, hilfreich*
- **If stuck in Teil 2:** say *"Was meinst du dazu? / Hast du noch eine Idee?"* to keep it going
- **Teil 1A is the only fully predictable part** — practise it until it flows naturally

## 8. Grammar Syllabus

All grammar topics that appear in the exam, in priority order based on frequency in Sprachbausteine, Schreiben, and Sprechen.

### Tier 1 — Directly tested in Sprachbausteine gap-fill (highest priority)

| # | Topic | Key points to know |
|---|-------|--------------------|
| 1 | **Perfekt** | haben/sein + Partizip II; strong verbs; separable verbs (ge- in middle); -ieren verbs (no ge-) |
| 2 | **Präteritum** | war, hatte, musste, konnte, wollte, durfte, sollte — these forms only, not full conjugation tables |
| 3 | **Modal verbs (Präsens)** | können, müssen, dürfen, sollen, wollen, möchten — conjugation + infinitive at end of sentence |
| 4 | **Word order** | V2 rule in main clause; verb-to-END after weil/dass/wenn/ob/obwohl/bevor/nachdem/während |
| 5 | **Prepositions + cases** | Always-Akk list; always-Dat list; two-way (Wo→Dat / Wohin→Akk) |
| 6 | **Subordinating conjunctions** | weil, dass, wenn, als, ob, obwohl, bevor, nachdem, während, seit(dem), damit |

### Tier 2 — Writing tasks, speaking, reading comprehension

| # | Topic | Key points to know |
|---|-------|--------------------|
| 7 | **Konjunktiv II** | würde + inf; wäre, hätte, könnte, müsste, dürfte — polite requests and hypotheticals |
| 8 | **Passive voice** | wird gemacht (Präsens); wurde gemacht (Präteritum); muss/kann gemacht werden |
| 9 | **Articles + cases (full)** | Nom/Akk/Dat endings for der/ein/kein; masculine accusative (den/einen); all Dativ forms |
| 10 | **Separable + reflexive verbs** | Prefix splits to end; ge- in middle for Partizip II; sich + Akk/Dat pronoun |
| 11 | **Verbs with fixed prepositions** | warten auf (Akk), sich freuen auf (Akk), denken an (Akk), sprechen mit (Dat), sich kümmern um (Akk), sich interessieren für (Akk) |

### Tier 3 — Richer writing and reading comprehension

| # | Topic | Key points to know |
|---|-------|--------------------|
| 12 | **Relative clauses** | der/die/das as relative pronoun; verb to end; case depends on function in relative clause |
| 13 | **Adjective endings** | After der/ein; Nom/Akk/Dat singular; -e vs -en pattern |
| 14 | **Infinitive with zu** | versuchen zu, anfangen zu, aufhören zu; um...zu (purpose); ohne...zu (without doing) |
| 15 | **Personal + reflexive pronouns** | mich/mir, dich/dir, sich; knowing when to use Akk vs Dat form |
| 16 | **Two-part connectors** | nicht nur...sondern auch; entweder...oder; zwar...aber; sowohl...als auch; weder...noch |

## 10. Exam Traps & Real Tips

From real exam accounts and prep teachers — not in the official handbook.

### General

**The perfectionist trap**
Many candidates get stuck on one hard question for 5–10 minutes and run out of time for easy ones at the end.
*Fix: If you don't know after 30 seconds, make your best guess and move on. You only need ~70% to pass.*

**Answer sheet trap**
You write answers in the question booklet but ONLY the Antwortbogen S30 is marked.
*Fix: Leave the last 5 minutes to transfer all answers carefully. Use pencil so you can erase.*

**Vocabulary panic in reading and listening**
Texts sometimes include vocabulary above B1 on purpose.
*Fix: You never need to understand every word. Focus on the question and find the relevant section.*

---

### Lesen traps

- Wrong options are often **true** about the text — but they don't answer the *specific question* asked
- In Teil 1: two ads look very similar — one tiny detail (age limit, day, what is offered) makes only one correct
- In Teil 3: read the **purpose** of the email (what does the sender want?), not just the topic
- Finish reading early? Use extra time for Sprachbausteine — experienced candidates finish 15–20 min early

---

### Sprachbausteine traps

- In Teil 1: all three options may look grammatically possible — choose the one that fits the **meaning and context**
- In Teil 2 word bank: 5 extra words are there to confuse — always check case and grammar, not just meaning
- Preposition + case is the most frequently lost point: *mit dem Bus / auf den Tisch / seit einer Woche*
- For the email writing (Teil 3): missing one content point loses marks — underline all required points first

---

### Hören traps

- **Teil 3 (announcements) is heard ONCE only** — the most commonly missed fact among candidates
- Speakers self-correct numbers: *"um 18 Uhr — nein, um 18:30"* → always take the **second value**
- For richtig/falsch: the statement must be **completely** correct — one wrong detail = falsch
- For the written note in Teil 5: write key information only (name, time, number) — not full sentences
- For the short reply: answer **exactly** the question asked (wann? wo? was?), not a general sentence
- Do NOT try to translate everything while listening — focus only on what the question needs

---

### Schreiben traps

- Missing a content point in the email = losing marks — underline everything you must address before writing
- Writing 200+ words wastes time and usually introduces more mistakes — 80–100 words is enough
- Forgetting verb-to-end after *weil/dass* is the most common grammar error in writing tasks
- Not opening or closing the email properly loses easy marks — these phrases should be automatic

---

### Sprechen traps

- **Short correct answers score lower than longer natural ones with small mistakes** — speak more, not less
- **Going silent in Teil 2**: if you run out of ideas say *"Was meinst du? / Hast du noch eine Idee?"*
- **Using the same words repeatedly**: instead of always *gut* try *toll, prima, sinnvoll, praktisch, hilfreich*
- **Not reacting to your partner**: say *"Ach wirklich? Das finde ich auch / Das sehe ich etwas anders."*
- **Memorising a monologue for Teil 3**: it must sound like a real conversation, not a prepared speech
- **Not self-correcting**: if you notice a mistake, correct it out loud — examiners value this

## 11. 15-Day Study Plan (May 5 → May 19)

### The strategy behind this plan

This is not one grammar topic per day. That approach wastes time on things you may already know and leaves no time to practise actual exam tasks.

The plan has three phases:

- **Phase 1 (Days 1–2): Diagnose.** Do the mock exam first. Find out exactly what you already know and what you are losing points on. This determines where you spend the most time.
- **Phase 2 (Days 3–11): Fix.** Grammar review + daily exam practice on every module. Vocabulary runs every day, not just one day.
- **Phase 3 (Days 12–15): Perform.** Mock exam, targeted fixes, speaking drills, final rest.

**Every single day includes:**
- 20 min listening practice (YouTube "Telc B1 hören test" — do 1–2 per day, every day)
- Vocabulary review for 10 min (rotate through the topic list in Section 9)

These two things are non-negotiable daily habits. Listening every day is the single most effective thing you can do to move your score.

### What high scorers actually did (from real exam accounts)

- **Mock exam first, not last.** Doing the mock on Day 1 instead of Day 15 gives you 2 weeks to fix the right things instead of guessing.
- **Listening every single day.** Not one day. Every day. 2–3 short Hören exercises per day builds the stamina and ear training that cannot be crammed the night before.
- **Speaking out loud, alone.** Write scripts for Teil 1A, Teil 2 scenarios, and Teil 3 opinion topics. Then practise them out loud — not silently. Speaking to yourself feels strange but works.
- **Connectors over grammar rules.** Using *deshalb, obwohl, trotzdem, außerdem* fluently in Schreiben and Sprechen signals B1 level to examiners more than knowing an obscure grammar rule.
- **One or two B1+ phrases in speaking.** Phrases like *"Ich bin der Meinung, dass..."* or *"Einerseits... andererseits..."* dropped naturally into conversation can push your speaking score higher than your actual level.
- **Finish reading fast, use the buffer.** Strong candidates finish Lesen in 25–30 minutes and use the remaining 15–20 minutes for Sprachbausteine. Time management in the reading block is a real score multiplier.
- **Write the answer sheet in pencil.** You can erase and correct. Transfer answers carefully — only the Antwortbogen S30 is marked.

## 13. Quick Reference Tables

### Article endings — Nominativ / Akkusativ / Dativ

| Case | Maskulin | Feminin | Neutrum | Plural |
|------|---------|---------|---------|--------|
| **Nom** | der / ein / kein | die / eine / keine | das / ein / kein | die / — / keine |
| **Akk** | **den / einen / keinen** | die / eine / keine | das / ein / kein | die / — / keine |
| **Dat** | **dem / einem / keinem** | **der / einer / keiner** | **dem / einem / keinem** | **den / — / keinen** |

> **Plural Dativ:** add **-n** to the noun if it doesn't already end in -n or -s
> Example: die Kinder → den Kindern | die Autos → den Autos (already -s, no change)

> **Key rule:** Only masculine changes in Akkusativ (der→den, ein→einen). Everything else stays the same as Nominativ except in Dativ.

### Prepositions — case rules

| Always Akkusativ | Always Dativ | Two-way prepositions |
|-----------------|-------------|----------------------|
| durch | mit | in, an, auf, über |
| für | bei | unter, vor, hinter |
| gegen | seit | neben, zwischen |
| ohne | von | |
| um | nach | |
| bis | aus | **Wo? (location) → Dativ** |
| entlang | zu | **Wohin? (direction) → Akkusativ** |
| | außer | |
| | gegenüber | |

**Two-way examples:**
- Das Buch liegt **auf dem** Tisch. (Wo? → Dat)
- Ich lege das Buch **auf den** Tisch. (Wohin? → Akk)
- Sie arbeitet **in der** Stadt. (Wo? → Dat)
- Sie fährt **in die** Stadt. (Wohin? → Akk)

| Verb | Konjunktiv II | English meaning |
|------|---------------|-----------------|

### Partizip II — formation rules

| Type | Rule | Examples |
|------|------|---------|
| Regular | ge- + stem + -t | ge**mach**t, ge**kauf**t, ge**spiel**t, ge**arbeit**et |
| Strong | ge- + changed stem + -en | ge**gess**en, ge**fahr**en, ge**schrieb**en, ge**nomm**en |
| Separable | prefix + **ge** + stem | auf**ge**macht, ein**ge**kauft, an**ge**rufen, mit**ge**nommen |

### haben vs. sein in Perfekt

| Use **sein** | Use **haben** |
|-------------|--------------|

| Change of state | Most verbs with a direct object (Akkusativ) |
| aufwachen, einschlafen, werden, aufstehen, sterben | |
| sein, bleiben, passieren | |
